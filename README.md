# 🗃️ Superstore SQL Analysis

SQL-based analysis of Superstore retail data using Python and SQLite.

## 🔍 Key Insights

- **Technology** generates the highest profit ($145K) despite Furniture having similar sales
- **West region** is the most profitable, while **Central** underperforms despite high sales volume
- **Copiers** have the highest profit margin with only 68 orders
- **Binders** are the biggest loss-maker with 613 loss-generating orders
- High discounts (40%+) result in **average loss of -$106 per order**

## 📊 SQL Queries Covered

| Query | Description |
|-------|-------------|
| Sales & Profit by Category | Compare revenue vs profitability |
| Profit by Region | Regional performance breakdown |
| Top Sub-Categories | Best performing product lines |
| Loss-making Orders | Identify unprofitable segments |
| Discount Impact Analysis | How discounts affect profit |
